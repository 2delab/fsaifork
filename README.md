# fsai_main

## Cloning

Clone the repo with all submodules in one command:

```bash
git clone --recurse-submodules https://github.com/MDXFormulaStudents-AI/fsai_workspace.git
```

## Already cloned?

If you cloned without `--recurse-submodules`, run this to pull in the submodules:

```bash
git submodule update --init
```

## Pulling updates

To pull the latest changes including any submodule updates:

```bash
git pull
git submodule update --init
```
